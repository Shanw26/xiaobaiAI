## 🚀 快速开始

### 第一步：配置 API Key

1. 点击左侧边栏底部的"设置"按钮
2. 选择模型厂商：
   - **Claude (Anthropic)**：使用 Claude 官方 API
   - **智谱 GLM**：使用国产平价模型（推荐新手）
3. 输入对应的 API Key
4. 选择模型（推荐：Claude 3.5 Sonnet 或 GLM-4.7）
5. 点击"保存"

### 获取 API Key

#### Claude API Key
1. 访问：https://console.anthropic.com/
2. 注册/登录账号
3. 在 API Keys 页面创建新的 API Key
4. 复制并粘贴到应用中

#### 智谱 GLM API Key
1. 访问：https://open.bigmodel.cn/
2. 注册/登录账号
3. 在 API Keys 页面创建新的 API Key
4. 复制并粘贴到应用中

### 第二步：开始对话

1. 点击左上角的"新对话"按钮
2. 在底部的输入框输入消息
3. 点击右侧的发送按钮（或按 Enter 键）
4. 等待 AI 响应（支持流式输出）

### 第三步：上传文件（可选）

1. 点击输入框左侧的 📎 按钮
2. 选择一个或多个文件
3. 文件会显示在输入框上方
4. 发送消息时，文件内容会自动发送给 AI

## 🎨 界面说明

## 💡 使用技巧

### 快捷键
- **Enter**：发送消息
- **Shift + Enter**：换行

### 对话管理
- 点击对话卡片切换对话
- 鼠标悬停在对话卡片上显示删除按钮
- 对话标题自动从第一条消息生成（前30字）

### 文件操作
- 支持多文件上传
- 支持的文件类型：
  - 文本文件：.txt, .md, .js, .jsx, .ts, .tsx, .py, .json, .csv, .html, .css
  - 图片文件：.png, .jpg, .jpeg, .gif, .webp, .bmp, .svg
- 文件内容会自动读取并发送给 AI

## 🐛 故障排除

### 应用无法启动
```bash
# 清理并重新安装依赖
rm -rf node_modules package-lock.json
npm install --cache /tmp/npm-cache-temp
npm run dev
```

### API Key 无效
- 检查 API Key 是否正确复制
- 确认 API Key 是否有效且未过期
- 检查账户余额是否充足

### AI 响应失败
- 确认网络连接正常
- 检查 API Key 权限
- 尝试切换不同的模型

### 文件无法读取
- 确认文件路径正确
- 检查文件权限
- 确认文件类型支持
